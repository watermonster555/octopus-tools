隐私政策

我们高度重视您的隐私保护

🔒 核心承诺
Octopus 所有工具均采用纯前端技术实现，您的数据仅在浏览器本地处理， 不会上传到任何服务器。我们承诺不收集、不存储、不分享您的任何个人信息和使用数据。

📋 我们收集的信息

我们不收集任何信息。

当您使用 Octopus 的任何工具时，所有操作都在您的浏览器本地完成：

  • 您上传的图片仅在浏览器中进行压缩处理，不会上传到服务器
  • 您输入的文本仅在浏览器中进行转换，不会被记录或传输
  • 您生成的密码完全在本地生成，我们不会保存任何副本
  • 您粘贴的 JSON 数据仅在本地格式化，不会离开您的设备

💾 数据存储

我们不存储任何数据。

Octopus 不会使用 Cookie、LocalStorage 或其他任何方式在您的设备上存储数据。 每次刷新页面后，所有临时数据都会被清除。

唯一的例外是密码生成器的历史记录功能，但这仅在您主动关闭隐私模式时才会启用， 且数据仅保存在当前页面内存中，关闭页面后立即清除。

🔗 第三方服务

Octopus 使用以下第三方服务：

  • GitHub Pages - 用于网站托管，仅提供静态文件服务
  • CDN - 用于加载部分开源 JavaScript 库（如二维码生成库）

这些服务仅提供网站托管和静态资源加载，不会接触到您在使用工具时处理的任何数据。

🛡️ 安全建议

虽然 Octopus 本身不收集任何数据，但我们仍建议您：

  • 使用密码生成器时，建议开启隐私模式，生成的密码不会显示在历史记录中
  • 处理敏感图片时，可以放心使用，图片数据不会离开您的设备
  • 定期更新浏览器，确保获得最新的安全保护
  • 在公共设备上使用后，建议关闭浏览器标签页

📝 政策更新

我们可能会不时更新本隐私政策。任何更改都会在本页面发布，并更新"最后更新日期"。 建议您定期查看本页面，了解我们如何保护您的信息。

如果您对本隐私政策有任何疑问或建议，欢迎通过 GitHub 与我们联系。
